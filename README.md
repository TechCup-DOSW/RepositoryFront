The **frontend branch** contains the client-side implementation of the TECHCUP Football Tournament Management System. This part of the project is responsible for providing the user interface that interacts with the backend API.

The frontend allows users to perform actions such as registering and logging into the platform, creating and managing teams, viewing tournament information, checking match schedules and results, and reviewing standings and statistics.

This branch focuses on delivering a responsive and user-friendly interface that communicates with the backend services through RESTful APIs.
